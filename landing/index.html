React Canvas Component

Install LetterGlitch and drop a living text field into your landing page.

A fast canvas-based background effect for React that cycles characters, blends between palette colors, and adds optional vignette overlays.

Install It See Usage
  • Typed for React + TypeScript
  • Canvas-rendered
  • Palette-driven glitch motion
Live preview LetterGlitch.tsx

Install

1. Add the component file

Save your React component as components/LetterGlitch.tsx. It uses only React hooks and the browser canvas API. 

components/LetterGlitch.tsx

Dependencies

2. Make sure your app has React and Tailwind utilities

The component itself needs React. The wrapper markup in your snippet uses Tailwind utility classes like relative, w-full, and bg-black. 

npm install react react-dom
npm install -D tailwindcss

Usage

3. Mount it full-screen or inside a hero block

import LetterGlitch from './components/LetterGlitch';

export default function Hero() {
  return (
    <section className="relative min-h-screen overflow-hidden bg-black text-white">
      <div className="absolute inset-0">
        <LetterGlitch
          glitchColors={['#2b4539', '#61dca3', '#61b3dc']}
          glitchSpeed={50}
          centerVignette={false}
          outerVignette
          smooth
        />
      </div>

      <div className="relative z-10 mx-auto flex min-h-screen max-w-5xl items-center px-6">
        <div>
          <p className="mb-4 text-sm uppercase tracking-[0.3em] text-emerald-300">
            Ambient canvas texture
          </p>
          <h1 className="max-w-3xl text-5xl font-semibold leading-tight">
            Build a hero section that already feels alive.
          </h1>
        </div>
      </div>
    </section>
  );
}

Props

4. Tune the effect

glitchColorsArray of hex colors used for letter swaps.
glitchSpeedMilliseconds between glitch updates.
centerVignetteAdds a darker radial center overlay.
outerVignetteAdds a dark edge falloff around the canvas.
smoothInterpolates between colors instead of snapping.
charactersCharacter pool used when generating the field.
